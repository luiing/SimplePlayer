# SimplePlayer
    AD Player simulate Tmall(广告播放器，模仿天猫，支持全屏，支持在线和离线播放，支持RecyclerView)

### 手机自带的MediaPlayer，支持http,https,rtsp

### 播放器只有唯一一组MediaPlayer和Surface在工作,切换视频会复用，内存消耗非常小

# Use
    compile 'com.uis:lib-simpleplayer:0.0.6'
